# Internship & Project Tracker

A simple and practical full-stack web application built to help students track their internships and projects in one place.

## Features
- Add and manage internship applications
- Add and track personal or academic projects
- View, edit, and delete entries easily

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
